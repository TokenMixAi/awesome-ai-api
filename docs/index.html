Why this exists

  • OpenRouter /models only lists gateways inside OpenRouter's own graph.
  • helpaio.com publishes human reviews, no machine-verified data.
  • apicompare.best compares prices but not authenticity or survival.
  • We do all three, and we cover the Chinese 中转站 ecosystem that Western trackers miss.

Methodology (TL;DR)

  1. Fetch each URL with a public bot UA.
  2. Classify against a bilingual relay-keyword lexicon.
  3. Probe /v1/models and /api/v1/models. A JSON models list, or 401/403 with auth hints, = real API.
  4. Fingerprint the OSS engine (new-api, one-api, dify, litellm, openrouter, …).
  5. Score = reachability + API confirmation + model count + uptime − latency penalty.
